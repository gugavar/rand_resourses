ამოცანა 1 — რიცხვითი ცვლადები და გამოთვლები

დაადეკლარით შემდეგი ცვლადები:

  • პროდუქტის მასა: m = 12.5 კგ
  • სითბოტევადობა: cp = 3.8 კჯ/(კგ·°C)
  • ტემპერატურის ცვლილება: dT = 45 °C
Q = m cp ΔT
შესასრულებელი
  1. გამოთვალეთ Q
  2. გამოიტანეთ შედეგი disp ბრძანებით

ამოცანა 2 — სიმბოლური გამოსახულების შექმნა

სიმბოლურად განსაზღვრეთ ფუნქცია:

f(x) = 3x2 - 5x + 7
შესასრულებელი
  1. შექმენით სიმბოლური გამოსახულება
  2. გაამარტივეთ ან გადაამოწმეთ, თუ საჭიროა
  3. იპოვეთ ფუნქციის მნიშვნელობა:
    • x = 0-ზე
    • x = 2-ზე
    • x = -1-ზე

ამოცანა 3 — სიმბოლური განტოლების ამოხსნა

ამოხსენით განტოლება:

x2 - 7x + 10 = 0
შესასრულებელი
  1. გამოიყენეთ solve
  2. გადაამოწმეთ შედეგი მიღებული ფესვების განტოლებაში ჩასმით
  3. მიუთითეთ, ფესვები ნამდვილია თუ კომპლექსური

ამოცანა 4 — განტოლებათა სისტემა

ამოხსენით სისტემა:

x + y = 12
2x - y = 3
შესასრულებელი
  1. გამოიყენეთ სიმბოლური ცვლადები
  2. ამოხსენით სისტემა
  3. MATLAB-ში გადაამოწმეთ ამონახსნი

ამოცანა 5 — წარმოებულები

მოცემულია ფუნქცია:

f(x) = x3 - 6x2 + 9x + 1
შესასრულებელი
  1. იპოვეთ პირველი წარმოებული
  2. იპოვეთ მეორე წარმოებული
  3. იპოვეთ პირველი წარმოებულის მნიშვნელობა x = 2-ზე
  4. მოკლედ განმარტეთ, რას აღწერს წარმოებული

ამოცანა 6 — ნაწილობრივი წარმოებულები

მოცემულია ფუნქცია:

f(x,y) = x2y + 4xy2
შესასრულებელი
  1. იპოვეთ ∂f/∂x
  2. იპოვეთ ∂f/∂y
  3. გამოთვალეთ ორივე (1,2) წერტილში

ამოცანა 7 — განუსაზღვრელი და განსაზღვრული ინტეგრალი

მოცემულია:

f(x) = 2x2 + 3x + 1
შესასრულებელი
  1. იპოვეთ განუსაზღვრელი ინტეგრალი
  2. იპოვეთ განსაზღვრული ინტეგრალი x = 0-დან x = 3-მდე
  3. განმარტეთ, რას ნიშნავს ეს ინტეგრალი გეომეტრიულად

ამოცანა 8 — რიცხვითი ფუნქციის გრაფიკი

შექმენით ვექტორი 0-დან 10-მდე ნაბიჯით 0.1 და ააგეთ ფუნქციის გრაფიკი:

y = e-0.4x
მოთხოვნები
  1. გამოიყენეთ plot
  2. დაამატეთ სათაური, ღერძების სახელები და ბადე
  3. მიუთითეთ, ფუნქცია ზრდადია თუ კლებადი

ამოცანა 9 — რამდენიმე ფუნქციის ერთ გრაფიკზე აგება

ერთსა და იმავე ფიგურაზე ააგეთ:

y1 = x
y2 = x2

ინტერვალზე 0 ≤ x ≤ 5

მოთხოვნები
  1. გამოიყენეთ ერთი plot ბრძანება ან hold on
  2. დაამატეთ legend
  3. დაამატეთ ღერძების სახელები და grid
  4. მოკლედ შეადარეთ გრაფიკების ფორმა

ამოცანა 10 — სიმბოლური ფუნქციის გრაფიკი

გამოიყენეთ fplot შემდეგი ფუნქციისთვის:

f(x) = x3 - 3x

ინტერვალზე [-3, 3]

შესასრულებელი
  1. დაამატეთ სათაური და ღერძების სახელები
  2. დაახლოებით დაადგინეთ, სად კვეთს გრაფიკი x-ღერძს
  3. შეადარეთ შედეგი solve-ის მიერ მიღებულ ფესვებს

ამოცანა 11 — ტენიანობის კლების მოდელი

მოცემულია გაშრობის მარტივი მოდელი:

M(t) = M0 e-kt

სადაც:

  • M0 = 80
  • k = 0.25
შესასრულებელი
  1. განსაზღვრეთ მოდელი სიმბოლურად
  2. ჩასვით რიცხვითი მნიშვნელობები
  3. იპოვეთ M(5)
  4. იპოვეთ dM/dt
  5. განმარტეთ, იზრდება თუ მცირდება ტენიანობა დროთა განმავლობაში

ამოცანა 12 — ტემპერატურის მოდელი

მოცემულია:

T(t) = 25 + 60e-0.3t
შესასრულებელი
  1. ააგეთ T(t)-ის გრაფიკი ინტერვალზე 0 ≤ t ≤ 20
  2. იპოვეთ წარმოებული სიმბოლურად
  3. გამოთვალეთ წარმოებული t = 0-ზე
  4. განმარტეთ წარმოებულის ნიშნის ფიზიკური მნიშვნელობა

ამოცანა 13 — კრიტიკული წერტილები

მოცემულია:

f(x) = x3 - 3x2 - 9x + 2
შესასრულებელი
  1. იპოვეთ პირველი წარმოებული
  2. ამოხსენით f′(x) = 0
  3. იპოვეთ მეორე წარმოებული
  4. დაახასიათეთ კრიტიკული წერტილები: ლოკალური მაქსიმუმია თუ ლოკალური მინიმუმი

ამოცანა 14 — ფართობი მრუდის ქვეშ

გამოთვალეთ MATLAB-ში:

∫04 (x2 + 1) dx
შესასრულებელი
  1. იპოვეთ ზუსტი მნიშვნელობა სიმბოლურად
  2. ააგეთ ფუნქციის გრაფიკი [0,4]-ზე
  3. განმარტეთ, რას ნიშნავს მიღებული ფართობი გეომეტრიულად

ამოცანა 15 — მაგალითი: სითბო და ტემპერატურა

მოცემულია მარტივი სითბური მოდელი:

Q(T) = 5T2 - 20T + 15
შესასრულებელი
  1. განსაზღვრეთ Q(T) სიმბოლურად
  2. იპოვეთ მისი წარმოებული
  3. იპოვეთ T-ს მნიშვნელობა(ები), სადაც dQ/dT = 0
  4. ააგეთ Q(T)-ის გრაფიკი ინტერვალზე 0 ≤ T ≤ 10
  5. მიუთითეთ, სად აქვს ფუნქციას მინიმუმი ან მაქსიმუმი

ამოცანა 16 — შერეული სიმბოლური და რიცხვითი სამუშაო

მოცემულია:

f(x) = (x2 + 1) / (x + 2)
შესასრულებელი
  1. განსაზღვრეთ ფუნქცია სიმბოლურად
  2. გაამარტივეთ, თუ შესაძლებელია
  3. იპოვეთ წარმოებული
  4. ააგეთ გრაფიკი ინტერვალზე [-1, 5]
  5. გამოთვალეთ ფუნქციის მნიშვნელობა x = 0, x = 1, x = 3-ზე
