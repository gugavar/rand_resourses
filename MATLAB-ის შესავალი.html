  • რიცხვით და სიმბოლურ გამოთვლებს შორის განსხვავების გააზრება

MATLAB-ს გამოვიყენებთ როგორც საინჟინრო ხელსაწყოს, რომელიც მათემატიკური მოდელის აღწერის, რიცხვითი გამოთვლების ჩატარების, შედეგის ვიზუალიზების და მისი ინტერპრეტირების საშუალებას გვაძლევს.

1. რა არის MATLAB?

MATLAB არის პროგრამული გარემო, რომელიც გამოიყენება:

  • რიცხვითი გამოთვლებისთვის
  • სიმბოლური გამოთვლებისთვის
  • გრაფიკების ასაგებად
  • განტოლებების ამოსახსნელად
  • მონაცემების დასამუშავებლად
  • პროგრამებისა და სკრიპტების დასაწერად

2. MATLAB-ის სამუშაო გარემო

2.1 Command Window

აქ ბრძანებებს პირდაპირ ვწერთ და შედეგს მაშინვე ვიღებთ.

მაგალითი
2 + 3
შედეგი
ans = 5

2.2 Script File

Script არის ფაილი, სადაც შეგვიძლია რამდენიმე ბრძანება ერთად შევინახოთ. ეს განსაკუთრებით გამოსადეგია როდესაც რამდენიმე გამოთვლის თანმიმდევრობით ჩატარებაა აუცილებელი.

მაგალითი
a = 5; b = 3; c = a + b; disp(c)

3. რიცხვითი ცვლადები

3.1 ცვლადის დეკლარირება

x = 5; y = 2.3; z = x * y;

3.2 ცვლადის სახელის შერჩევის წესები

  • უნდა იწყებოდეს ასოთი
  • შეიძლება შეიცავდეს ციფრებს
  • შეიძლება შეიცავდეს ქვედა ტირეს _
  • არის case-sensitive
მაგალითი
T = 50; t = 10;

აქ T და t სხვადასხვა ცვლადებია.

4. ძირითადი არითმეტიკული ოპერაციები

a + b % შეკრება a - b % გამოკლება a * b % გამრავლება a / b % გაყოფა a^2 % ხარისხში აყვანა sqrt(a) % კვადრატული ფესვი exp(a) % e^a log(a) % ნატურალური ლოგარითმი

5. სიმბოლური ცვლადები — syms

როცა გვინდა MATLAB-მა იმუშაოს ალგებრულ გამოსახულებებთან ვიყენებთ სიმბოლურ ცვლადებს.

syms x

ახლა x არის სიმბოლური ცვლადი.

მაგალითი
syms x f = x^2 + 3*x + 2

რამდენიმე სიმბოლური ცვლადის გამოცხადება:

syms x y t

ეს განსაკუთრებით გამოგვადგება:

  • განტოლებების ამოსახსნელად
  • წარმოებულებისთვის
  • ინტეგრალებისთვის
  • გამოსახულებების გასამარტივებლად

6. გამოსახულებების შექმნა და გარდაქმნა

syms x f = 2*x^3 - 5*x + 1;

გამოსახულებებთან გამოსადეგი ბრძანებებია:

expand(f) factor(f) simplify(f) collect(f)
მაგალითი
syms x f = (x + 1)^2; expand(f)
შედეგი
x^2 + 2*x + 1

7. მნიშვნელობების ჩასმა — subs

თუ გვინდა კონკრეტული რიცხვი ჩავანაცვლოთ სიმბოლურ გამოსახულებაში, ვიყენებთ subs-ს.

მაგალითი
syms x f = x^2 + 3*x + 1; subs(f, x, 2)

რამდენიმე ცვლადის ერთდროული ჩანაცვლება:

syms x y f = x^2 + y; subs(f, [x y], [3 4])

8. განტოლებების ამოხსნა

8.1 ერთი განტოლება

syms x solve(x^2 - 5*x + 6 == 0, x)

8.2 პარამეტრიანი განტოლება

syms x k solve(k*x + 2 == 0, x)

8.3 განტოლებათა სისტემა

syms x y eq1 = x + y == 5; eq2 = x - y == 1; solve([eq1, eq2], [x, y])
ყურადღება: განტოლების ჩაწერისას ამოხსნისთვის უნდა გამოიყენოთ == და არა =.

9. წარმოებულები

წარმოებული აღწერს, როგორ იცვლება ფუნქცია.

9.1 პირველი წარმოებული

syms x f = x^3 + 2*x^2 - x + 1; df = diff(f, x)

9.2 მეორე წარმოებული

d2f = diff(f, x, 2)

9.3 ნაწილობრივი წარმოებულები

syms x y f = x^2*y + sin(x*y); diff(f, x) diff(f, y)

ინჟინერიაში წარმოებული ხშირად აღწერს:

  • ცვლილების სიჩქარეს
  • დახრილობას
  • მაქსიმუმსა და მინიმუმს
  • დროზე, ტემპერატურაზე ან კონცენტრაციაზე დამოკიდებული სიდიდის ცვლილებას

10. ინტეგრირება

ინტეგრალებისთვის გამოიყენება int.

10.1 განუსაზღვრელი ინტეგრალი

syms x f = 3*x^2 + 2*x + 1; int(f, x)

10.2 განსაზღვრული ინტეგრალი

int(f, x, 0, 2)

10.3 ორმაგი ინტეგრალი

syms x y f = x + y; int(int(f, y, 0, 1), x, 0, 2)

ინტეგრალი ხშირად გვაძლევს:

  • დაგროვილ რაოდენობას
  • მთლიან მასას
  • მთლიან სითბოს
  • საშუალო მნიშვნელობას
  • ფართობს მრუდის ქვეშ

11. რიცხვითი გრაფიკების აგება

11.1 ძირითადი გრაფიკი — plot

x = 0:0.1:10; y = x.^2; plot(x, y)

ვექტორებთან მუშაობისას აუცილებელია ელემენტობრივი (წევრ-წევრი) ოპერატორების გამოყენება:

ოპერატორი დანიშნულება
.* ელემენტობრივი გამრავლება
./ ელემენტობრივი გაყოფა
.^ ელემენტობრივი ხარისხი
მაგალითი
y = x.^2 + 2*x + 1;

11.2 წარწერები და ბადე

plot(x, y) xlabel('x') ylabel('y') title('y = x^2') grid on

11.3 რამდენიმე ფუნქცია ერთ გრაფიკზე

x = 0:0.1:10; y1 = x; y2 = x.^2; plot(x, y1, x, y2) legend('y = x', 'y = x^2') grid on

12. სიმბოლური ფუნქციის გრაფიკი — fplot

თუ ფუნქცია სიმბოლურად გვაქვს მოცემული ვიყენებთ fplot ბრძანებას.

syms x f = x^3 - 4*x; fplot(f, [-3 3]) grid on xlabel('x') ylabel('f(x)') title('სიმბოლური ფუნქციის გრაფიკი')
fplot თავად არჩევს გრაფიკის ასაგებად საჭირო წერტილებს, ამიტომ სიმბოლურ ფუნქციებთან ხშირად უფრო მოსახერხებელია, ვიდრე ჩვეულებრივი plot.

13. სიმბოლურიდან რიცხვით ფუნქციაზე გადასვლა

ხშირად გვინდა სიმბოლურად შექმნილი გამოსახულება გადავაქციოთ ჩვეულებრივ MATLAB ფუნქციად.

syms x f = x^2 + 1; fh = matlabFunction(f); fh(3)

ეს გამოსადეგია მაშინ, როცა სიმბოლურ ანალიზს ვაერთიანებთ რიცხვით გამოთვლებთან ან გრაფიკებთან.

14. საინჟინრო მაგალითი

ვთქვათ, ტენის შემცველობა დროზეა დამოკიდებული:

M(t) = M0 * exp(-k*t)

syms M0 k t M = M0*exp(-k*t);

წარმოებული:

diff(M, t)

ინტეგრალი:

int(M, t)

კონკრეტული მნიშვნელობების ჩასმა:

subs(M, [M0 k t], [80 0.3 5])
სტანდარტული საინჟინრო სქემა
  1. მოდელის ჩაწერა
  2. სიმბოლური ანალიზი
  3. რიცხვითი შეფასება
  4. გრაფიკული წარმოდგენა

16. ვექტორები და მასივები

16.1 მწკრივი ვექტორი

x = [1 2 3 4];

16.2 სვეტოვანი ვექტორი

17. ყველაზე გავრცელებული შეცდომები

17.1 წერტილ-მძიმის დავიწყება

a = 5;

თუ ; არ დავწერეთ, MATLAB შედეგს ეკრანზე გამოიტანს.

17.2 ^-ის გამოყენება .^-ის ნაცვლად

არასწორია:

y = x^2

სწორია:

y = x.^2

17.3 სიმბოლური და რიცხვითი ცვლადების არევა

syms x x = 5;

ამ შემთხვევაში სიმბოლური x შეიცვლება რიცხვითი მნიშვნელობით.

17.4 განტოლების არასწორად ჩაწერა

სწორია:

x^2 - 4 == 0

არასწორია:

x^2 - 4 = 0

  3. სწორად ჩაწერე გამოსახულებები
  4. გამოიყენე შესაბამისი ბრძანებები: solve, diff, int, subs, plot, fplot
  5. მოახდინე შედეგის ფიზიკური ან მათემატიკური ინტერპრეტაცია

19. მაგალითები

მაგალითი 1 — განტოლების ამოხსნა
syms x sol = solve(2*x + 3 == 7, x)
მაგალითი 2 — წარმოებულის პოვნა
syms x f = x^3 - 2*x + 1; df = diff(f, x)
მაგალითი 3 — ინტეგრალის გამოთვლა
syms x f = 4*x; I = int(f, x, 0, 3)
მაგალითი 4 — რიცხვითი გრაფიკის აგება
x = 0:0.1:5; y = exp(-x); plot(x, y) grid on
მაგალითი 5 — სიმბოლური გრაფიკის აგება
syms x fplot(sin(x), [0 2*pi]) grid on
